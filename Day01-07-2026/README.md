## Topics Covered

### 1. CSS Position Property
The session began with an introduction to the `position` property in CSS. Different positioning methods were discussed along with their practical applications.

The following position values were covered:

- Static
- Relative
- Absolute
- Fixed
- Sticky

The behavior and use cases of each positioning method were explained in detail.

---

### 2. Static Position

- Default position of HTML elements.
- Elements follow the normal document flow.
- `top`, `left`, `right`, and `bottom` properties do not affect static elements.

---

### 3. Relative Position

- Positions an element relative to its original position.
- Allows movement using `top`, `left`, `right`, and `bottom`.
- The original space occupied by the element remains reserved.

---

### 4. Absolute Position

- Positions an element relative to its nearest positioned parent.
- Removes the element from the normal document flow.
- Used for placing elements at exact locations.

---

### 5. Fixed Position

- Keeps an element fixed relative to the browser window.
- The element remains visible even while scrolling.
- Commonly used for navigation bars, buttons, and headers.

---

### 6. Sticky Position

- Combines the behavior of relative and fixed positioning.
- The element behaves like a normal element until a specified scroll position is reached.
- Once reached, it sticks to the specified position on the screen.

---

### 7. Z-Index

- Introduction to the `z-index` property.
- Used to control the stacking order of overlapping elements.
- Elements with higher `z-index` values appear above elements with lower values.
