# HTML Training - Day 1

## Overview
This repository contains the HTML programs completed on **Day 1** of Web Development Training. The exercises focus on learning the basics of HTML, including webpage structure, text formatting, and table creation.

## Programs

### 1. About Myself
- Created a simple personal profile webpage.
- Used headings, paragraphs, and text formatting tags.
- Displayed personal information, education, and skills.

### 2. Basic Table
- Created a table using:
  - `<table>`
  - `<thead>`
  - `<tbody>`
  - `<tfoot>`
- Learned the structure of HTML tables.

### 3. Employee Table
- Displayed employee details such as:
  - Employee Name
  - Age
  - Salary

### 4. Rowspan and Colspan Table
- Practiced merging rows using `rowspan`.
- Practiced merging columns using `colspan`.

### 5. Time Table
- Designed a school timetable.
- Used `rowspan` and `colspan` for table formatting.

### 6. Complex Table Layout
- Created an advanced table using multiple merged rows and columns.

## Concepts Covered
- HTML Document Structure
- Headings (`<h1>`)
- Paragraphs (`<p>`)
- Text Formatting (`<b>`, `<i>`, `<u>`)
- Line Break (`<br>`)
- Font Tag (`<font>`)
- Tables (`<table>`, `<tr>`, `<th>`, `<td>`)
- Table Sections (`<thead>`, `<tbody>`, `<tfoot>`)
- `rowspan` and `colspan`
- Text Alignment


## Learning Outcomes
- Understood the basic structure of an HTML document.
- Learned how to create and format web pages.
- Gained knowledge of HTML tables and their components.
- Practiced using `rowspan` and `colspan` for complex table layouts.
- Built a strong foundation for learning CSS and JavaScript.
