# MERN Stack Training - Daily Diary

## Day 5

**Date:** 30-06-2026

### Objective
The objective of today's training was to gain an in-depth understanding of the CSS Box Model and learn how different CSS properties such as margin, padding, border, border-radius, box-sizing, and float are used to design structured and visually appealing web pages. The session focused on creating different shapes, understanding spacing between elements, and designing layouts using nested containers.

---

## Topics Covered

### 1. CSS Box Model
- Introduction to the CSS Box Model.
- Components of the Box Model:
  - Content
  - Padding
  - Border
  - Margin
- Importance of the Box Model in webpage design.

### 2. Box-Sizing Property
- Understanding `box-sizing`.
- Difference between:
  - `content-box`
  - `border-box`
- Benefits of using `box-sizing: border-box`.

### 3. Margin Property
- Using margin to create space outside elements.
- Margin shorthand:
  - `margin: top right bottom left;`
- Individual margin properties:
  - `margin-top`
  - `margin-right`
  - `margin-bottom`
  - `margin-left`

### 4. Padding Property
- Creating space inside elements.
- Padding shorthand.
- Individual padding properties.

### 5. Border Property
- Border width
- Border style
- Border color
- Border shorthand
- Border styles:
  - Solid
  - Dotted
  - Dashed

### 6. Border Radius
- Rounded corners using `border-radius`.
- Different radius values for each corner.
- Creating circles using `border-radius: 50%`.

### 7. Float Property
- `float: left`
- `float: right`
- Positioning multiple elements side by side.

### 8. Nested Box Layout
- Creating multiple nested `<div>` elements.
- Visualizing Content, Padding, Border, and Margin.
- Understanding how nested containers are designed.

---

## Concepts Learned

- Every HTML element behaves like a rectangular box.
- Difference between margin, border, padding, and content.
- Importance of `box-sizing` in webpage layouts.
- Using margins for external spacing.
- Using padding for internal spacing.
- Applying borders using different styles.
- Creating different shapes using `border-radius`.
- Designing circles with CSS.
- Positioning elements using the `float` property.
- Building webpage layouts using nested containers.

## Skills Gained

By the end of today's session, I was able to:

- Use margin and padding to control spacing.
- Create borders with different styles and colors.
- Design rounded boxes and circles using `border-radius`.
- Arrange webpage elements using float.

- Understand how CSS properties affect webpage structure and appearance.
